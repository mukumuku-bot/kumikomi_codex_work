# kumikomi_codex1

スマホのカメラを使って人物を検知し、検知した人物の中心が画面中央からどのくらいズレているかを%で表示するWebアプリです。
画面上部の犬風の目が、検知した人物の位置を追いかけるように動きます。

## 仕組み

ブラウザでカメラ映像を取得し、TensorFlow.jsのCOCO-SSDモデルで `person` を検知します。複数人が映った場合は、信頼度が最も高い人物を基準にします。

ズレ率は、画面中央から人物の検出枠中心までの距離を画面幅・高さに対して正規化して表示します。

- 中心からのズレ: 横方向と縦方向を合わせた総合値
- 横方向: 左右どちらへ何%ズレているか
- 縦方向: 上下どちらへ何%ズレているか
- 検知人数: 画面内で検知した人数

音が入った瞬間に左右へ表示が振動しないよう、方向の切り替えには一定以上の差が短時間続くことを条件にしています。0付近の小さな揺れは「正面付近」として扱います。

## 使い方

1. ローカルサーバーでこのフォルダを配信します。
2. スマホで同じネットワーク上のURLを開きます。
3. 「カメラ開始」を押して、カメラの使用を許可します。
4. 人物が画面に入ると、検出枠とズレ率が表示されます。

## 起動例

```powershell
python -m http.server 8000
```

PCで開く場合:

```text
http://localhost:8000
```

スマホで開く場合は、PCのIPアドレスを使って次のようにアクセスします。

```text
http://PCのIPアドレス:8000
```

ブラウザの仕様により、スマホのカメラは `localhost` または HTTPS でないと使えない場合があります。スマホ実機ではHTTPS配信、または開発用トンネルの利用が安定します。

## 実装内容

- スマホの内カメラ固定
- TensorFlow.js + COCO-SSDによる人物検知
- 検出枠、人物中心、画面中心からの線を表示
- 中心からのズレを%で表示
- 犬風の目が人物の位置へ視線追従
- オフライン表示用の簡易Service Worker
