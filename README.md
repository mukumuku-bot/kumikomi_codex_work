# 音源方向推定と文字起こし

スマホのマイクを使って、音源の左右方向を推定しながら、同じ画面で音声の文字起こしもできるWebアプリです。

## 使い方

1. ローカルサーバーでこのフォルダを配信します。
2. スマホで同じネットワーク上のURLを開きます。
3. 音源方向推定を使う場合は「測定開始」を押して、マイクの使用を許可します。
4. 文字起こしを使う場合は「文字起こし開始」を押して話します。
5. 結果はコピー、テキスト保存、消去ができます。

## 起動例

```powershell
python -m http.server 8000
```

PCで開く場合:

```text
http://localhost:8000
```

スマホで開く場合は、PCのIPアドレスを使って次のようにアクセスします。

```text
http://PCのIPアドレス:8000
```

マイクや音声認識は、ブラウザの仕様により `localhost` または HTTPS でないと使えない場合があります。文字起こしは Web Speech API に対応したブラウザで動作します。

## 実装内容

- Web Audio APIでマイク音量を取得
- ChannelSplitterで左右チャンネルの音量差を計算
- 音源方向を左方向、正面付近、右方向として表示
- Web Speech APIでスマホから文字起こし
- 文字起こし結果のコピー、テキスト保存、消去
- オフライン表示用の簡易Service Worker
