# kumikomi_codex1

スマホの内カメラで人物を検知し、画面に表示した犬風の目だけが人物を追うWebアプリです。

画面をタップするとカメラを開始します。カメラ映像や数値パネルは表示せず、検知処理は裏側で動きます。

## 使い方

1. アプリのURLをスマホで開きます。
2. 画面をタップして、カメラの使用を許可します。
3. 人物が映ると、目がその方向へ追従します。

## 実装内容

- スマホの内カメラ固定
- TensorFlow.js + COCO-SSDによる人物検知
- SVGの目だけを画面に表示
- 人物位置への視線追従
- 自然な瞬き
- オフライン表示用の簡易Service Worker
